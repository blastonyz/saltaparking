SEM Inteligente Salta
Plataforma urbana inteligente para modernizar el Sistema de Estacionamiento Medido
1. Introducción
La presente propuesta busca modernizar integralmente el Sistema de Estacionamiento Medido (SEM) de la ciudad de Salta mediante una solución tecnológica inclusiva, rápida y basada en datos urbanos en tiempo real.
Actualmente el sistema funciona mediante talonarios físicos, pagos manuales y ausencia total de trazabilidad digital, lo que genera:
falta de control,
cobros indebidos,
evasión,
ausencia de datos operativos,
congestión urbana,
y una experiencia poco eficiente tanto para conductores como para permisionarios.
La solución propuesta transforma el SEM en una plataforma urbana inteligente donde:
todos los pagos quedan registrados digitalmente,
los conductores pueden pagar de manera simple,
la Municipalidad obtiene información en tiempo real,
y los permisionarios continúan siendo actores centrales del sistema.
La propuesta prioriza:
velocidad operativa,
inclusión social,
simplicidad,
trazabilidad,
y alta factibilidad técnica.

2. Visión general de la solución
La plataforma combina:
aplicación móvil,
geolocalización,
pagos digitales,
QR urbanos,
información colaborativa,
y validación humana.
El objetivo no es únicamente digitalizar el cobro del estacionamiento.
La verdadera innovación consiste en convertir al SEM en:
una plataforma inteligente de movilidad urbana en tiempo real.

4. Funcionamiento general del sistema
4.1 Ingreso a zona SEM
La plataforma utiliza geolocalización y geofencing para detectar cuando un conductor ingresa a una zona habilitada del SEM.
La aplicación puede mostrar:
“Ingresaste a una zona de estacionamiento medido.”
Esto permite iniciar automáticamente parte del flujo operativo.

4.2 Visualización de disponibilidad
El conductor puede visualizar desde la app:
zonas libres,
zonas con poca disponibilidad,
zonas congestionadas.
La disponibilidad se muestra mediante:
🟢 Disponible
🟡 Pocos lugares
🔴 Completo
El objetivo no es indicar slots exactos individuales, sino brindar información urbana útil y realista para reducir vueltas innecesarias y congestión.

4.3 Cómo se calcula la disponibilidad
La ocupación se construye mediante un modelo híbrido:
Información de permisionarios
Los permisionarios actualizan rápidamente:
disponible,
pocos lugares,
completo.

Geolocalización
La plataforma analiza:
entradas,
salidas,
permanencia,
movimiento vehicular.

Finalización automática
Cuando un vehículo abandona una zona:
el sistema infiere liberación de espacio.

5. Flujo completo de pago para conductores
La solución contempla distintos niveles tecnológicos para garantizar accesibilidad total.

5.1 Usuario con aplicación móvil
Paso 1
El conductor estaciona.

Paso 2
La app detecta:
permanencia,
velocidad cero,
ubicación válida.
La aplicación puede sugerir:
“¿Deseás iniciar estacionamiento?”

Paso 3
El conductor inicia la sesión con un toque.

Paso 4
Selecciona:
pago automático,
o pago manual.

Paso 5
El sistema calcula automáticamente:
tiempo utilizado,
fraccionamiento,
descuentos,
tolerancias.

Paso 6
Cuando el vehículo abandona la zona:
la sesión finaliza automáticamente,
o solicita confirmación.

5.2 Usuario sin aplicación móvil
Para garantizar inclusión tecnológica, cada cuadra cuenta con:
QR urbanos visibles.
El conductor puede:
escanear el QR,
ingresar patente,
seleccionar método de pago.
No es obligatorio descargar una app.

5.3 Pago digital
Si el usuario elige pago digital:
el sistema genera QR o link de Mercado Pago,
el conductor paga mediante:
transferencia,
débito,
crédito,
dinero en cuenta.
La validación es instantánea.
El descuento del 20 % previsto por ordenanza se aplica automáticamente.

5.4 Pago en efectivo
Si el usuario elige efectivo:
el sistema envía una alerta al permisionario responsable de la cuadra.
La app del permisionario muestra:
ubicación aproximada del vehículo,
patente,
solicitud de cobro.
El permisionario se acerca únicamente a vehículos que realmente requieren asistencia.
Esto elimina la necesidad de recorrer constantemente toda la cuadra.

6. Registro digital obligatorio
Todo pago queda registrado digitalmente, incluyendo efectivo.
Cada operación almacena:
patente,
horario,
duración,
monto,
ubicación,
método de pago,
permisionario,
timestamp.
Esto elimina completamente:
talonarios físicos,
cobros no registrados,
falta de trazabilidad.

7. Rol del permisionario en el nuevo sistema
La propuesta NO elimina a los permisionarios.
La solución los transforma en:
operadores urbanos digitales.

7.1 Nuevas funciones
Registro de operaciones
pagos digitales,
pagos en efectivo,
validaciones.

Supervisión de ocupación
Actualización rápida de disponibilidad de la cuadra.

Asistencia puntual
Intervención únicamente cuando:
existe solicitud de efectivo,
el conductor necesita ayuda,
se requiere validación manual.

Validación operativa
Visualización de:
vehículos activos,
operaciones pendientes,
estado de ocupación.

7.2 Cómo mejora su trabajo
Menos caminata
Ya no necesitan recorrer constantemente toda la cuadra.

Menos cálculos manuales
La app calcula automáticamente:
montos,
fraccionamientos,
horarios,
descuentos.

Menos efectivo
Más pagos digitales implican:
menos riesgo,
menos errores,
menos manejo físico de dinero.

Menos conflictos
El sistema aplica automáticamente:
horarios válidos,
zonas habilitadas,
reglas de ordenanza.

7.3 Inclusión y accesibilidad
Muchos permisionarios:
son adultos mayores,
jubilados,
o personas con limitaciones físicas o tecnológicas.
Por ello la aplicación prioriza:
botones grandes,
pocos pasos,
alto contraste,
mínima escritura,
aprendizaje rápido.
El sistema fue diseñado para ser aprendido en pocos minutos.

14. Conclusión
La propuesta transforma el Sistema de Estacionamiento Medido de Salta en una plataforma urbana inteligente, inclusiva y basada en datos en tiempo real.
El sistema:
elimina los talonarios físicos,
digitaliza completamente las operaciones,
mantiene a los permisionarios como actores centrales,
mejora la experiencia del conductor,
reduce congestión urbana,
y permite a la Municipalidad acceder por primera vez a información operativa instantánea.
La innovación principal no consiste solamente en digitalizar pagos, sino en construir un sistema híbrido donde tecnología, geolocalización y operadores humanos trabajen juntos para modernizar la movilidad urbana de manera realista, accesible y escalable.
